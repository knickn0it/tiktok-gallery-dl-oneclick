# 🎵 TikTok One-Click Downloader (gallery-dl)

True one-click Windows TikTok downloader.
Double-click → paste URL → download.

## Features
- No watermark
- Supports video / profile / hashtag
- Optional cookies support
- Beginner friendly

## Requirements
- Python (added to PATH)
- gallery-dl (`pip install -U gallery-dl`)

## Usage
1. Double-click `download.bat`
2. Paste TikTok URL
3. Press Enter
